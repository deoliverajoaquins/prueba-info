# Resumen generado con Claude: ¿Qué es una hoja de cálculo?

Una hoja de cálculo es un programa que organiza información en forma de tabla, con filas y columnas, como si fuera una cuadrícula gigante.  
Se usa principalmente para trabajar con números: hacer cálculos automáticos, crear presupuestos, llevar registros de gastos o inventarios, y analizar datos.  
Lo más útil es que puede hacer operaciones matemáticas por vos (sumas, promedios, porcentajes) y si cambias un número, todo se actualiza solo.  
También permite crear gráficos para visualizar la información de manera más clara.  
Los programas más conocidos son Excel, Google Sheets y LibreOffice Calc.  
Es una herramienta muy práctica tanto para uso personal como profesional.
